# Ecommerce Backend
## Description
allows backend to communicate with frontend to fetch and display data

## Installation
- run 'npm i' in terminal

## Usage
-run 'npm i' in terminal
- login to mysql 
- run "SOURCE schema.sql" in mysql terminal
- run "node seeds.sql" in terminal
- run "node index.js" in terminal
